What is shift? It’s how many positions each letter moves in the alphabet.
Example: Shift 3 → A→D, B→E, C→F…

Non-letter characters remain unchanged.

How does it work?

Crypti uses the classic Caesar Cipher — a simple substitution that rotates letters by a fixed amount.

1

Pick a Shift

Set a number from 1–25. This is how many positions each letter will move. Shift 3 turns A→D, B→E, C→F, and so on.

2

Click Encode

We take each letter in your message and move it forward by your chosen shift. Example: HELLO (Shift 3) becomes KHOOR.

3

Click Decode

Have an encoded message? Paste it, set the correct shift, and hit Decode. We move letters backward to reveal the original text.

Original: HELLO WORLD

Encoded (Shift 3): KHOOR ZRUOG

Decoded: HELLO WORLD

Good to know:

  • Spaces, punctuation, and numbers stay the same.
  • Uppercase/lowercase can be preserved via script logic.
  • This is for education/fun — not cryptographically secure.
